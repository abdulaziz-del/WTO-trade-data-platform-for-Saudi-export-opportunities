GATT 1994 — Most Favoured Nation
FAVORABLE
Article I:1 GATT ensures Saudi exports receive equal treatment as most-favored-nation. 12 monitored markets confirmed MFN compliance for HS chapters 27, 28, 31. No discriminatory tariff detected.
TBT Agreement — Technical Barriers
ACTION NEEDED
Art. 2.2 TBT — EU notification G/TBT/N/EU/892 introduces new labeling requirements for chemical products. HS 2815 exporters must review compliance by 2025-09-01. Estimated adjustment cost: moderate.
SPS Agreement — Food Safety Measures
MONITOR
Japan SPS measure G/SPS/N/JPN/503 sets new MRL for food products HS 2106. CODEX Alimentarius basis confirmed under Art. 3 SPS. Saudi food exporters advised to verify conformity.
CBAM — Carbon Border Adjustment
HIGH RISK
EU CBAM under DS613 review. Potential GATT Art. III violation. Petrochemicals, cement, fertilizers from Saudi Arabia directly affected. Recommend proactive DS monitoring and stakeholder engagement.

Setup & Deployment Guide

Step-by-step production deployment for WTO Trade Intelligence Platform

🚀 Quick Start
1
Clone & Install Dependencies
Clone the repository and install all Node.js and Python dependencies.
git clone https://github.com/org/wto-intel-platform
cd wto-intel-platform
npm install          # Backend
cd frontend && npm install  # Frontend
2
Configure Environment Variables
Copy .env.example and set all required variables.
cp .env.example .env
# Set in .env:
DATABASE_URL=postgresql://user:pass@db:5432/wto
REDIS_URL=redis://cache:6379
ANTHROPIC_API_KEY=sk-ant-...
EPING_API_KEY=...
JWT_SECRET=...
AWS_REGION=me-south-1
3
Initialize Database
Run migrations and seed with sample WTO data.
npm run db:migrate
npm run db:seed          # Load mock data
npm run elasticsearch:init  # Create indices
4
Start Services (Docker Compose)
Launch all microservices, databases, and queues.
docker-compose up -d
# Services started:
# • API Gateway      :3000
# • Frontend         :3001
# • PostgreSQL       :5432
# • Elasticsearch    :9200
# • Redis            :6379
5
Run Initial WTO Data Ingestion
Trigger first-time sync of WTO notifications and tariff data.
npm run ingest:wto        # TPR Reports
npm run ingest:eping      # TBT/SPS
npm run ingest:tariffs    # Tariff schedules
npm run analyze:batch     # AI analysis run
6
Deploy to AWS (Production)
Push to GitHub to trigger CI/CD pipeline to AWS ECS in Riyadh region.
git push origin main
# GitHub Actions auto-deploys:
# build → test → docker push → ECS deploy
# Target: AWS me-south-1 (Riyadh)
